# 🎨 Sass Custom Grid System

A **fully customizable grid system** built with **Sass**, allowing developers to create responsive layouts with ease. This project serves as an alternative to traditional CSS frameworks by providing a flexible and lightweight grid system.

## 🎨 About This Project
This project is a **responsive grid system** developed using **Sass**. It provides a simple yet powerful way to build custom layouts with CSS Grid and Flexbox.

## ✨ Features:
- 🎯 **Customizable Grid System** with `Sass` variables  
- 📱 **Responsive Layout** using `row-cols` and `gutter spacing`  
- ⚡ **Lightweight & Fast** compared to Bootstrap/Grid frameworks  
- 🛠 **Easy to Use** with class-based grid structure  
- 🎨 **Built with Modern CSS & Sass best practices**

🚀 How to Run Locally
Open Terminal (macOS/Linux) or Command Prompt (Windows) and run:

1. **Clone this repository:**
```sh
    git clone https://github.com/thanhhang31023/sass-custom-grid-system.git
```
2. **Navigate to the project directory:**
```sh
    cd sass-custom-grid-system
```
3. **Install dependencies (if needed):**
   ```sh
    npm install
   ```
4. **Start the project with Sass compilation:**
```sh
    npm run start
```
5. **Open file index.html in your browser.**

Windows: Open File Explorer, navigate to the project folder, hold Shift + Right Click, then select "Open with Browser".
macOS/Linux: Use:
```sh
    open index.html  # macOS
    xdg-open index.html  # Linux
```
Or, manually drag and drop index.html into your browser.
